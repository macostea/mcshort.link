# Shorty McShort Link
This is a basic URL shortener. It does nothing more, nothing less.

## Requirements
### Frontend
* Node.js >= 16.10.0

### Backend
* Emscripten 2.0.31-git
* [wrangler](https://developers.cloudflare.com/workers/cli-wrangler/install-update)

## Installation
### Frontend
* Install npm dependencies
```shell script
$ npm install
```

* Run the application
```shell script
$ npm run dev
```

### Backend
* Run the application
``` shell script
$ wrangler dev
```
